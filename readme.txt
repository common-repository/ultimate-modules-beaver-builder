== Description ==

[Ultimate Modules - Beaver Builder](http://www.wpexpertdeveloper.com/ultimate-modules-beaver-builder/) is an awesome collection of UI and functional elements for Beaver Builder. Includes image sliders, galleries, image viewers.

This is an addon for Beaver Builder. You need to install Beaver Builder version 2.0.3.2 or higher before using this plugin.

More elements coming soon!!!!


> **Features**
>
> * Simple and Flexible Image Slider elements
> * Image Gallery elements
> * Image Viewer with Zooming, Rotation, Scaling
> * 10 Awesome Slide Transition Effects

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ultimate-modules-beaver-builder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use Ultimate Modules Beaver Builder to create amazing components.
4. Follow the <a href="http://www.wpexpertdeveloper.com/ultimate-modules-beaver-builder/">Documentation and Usage</a>.
